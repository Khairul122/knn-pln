PLN GridRisk — Sistem Prediksi Risiko Jaringan Distribusi

PLN GridRisk adalah aplikasi berbasis web yang dirancang untuk menganalisis dan memprediksi tingkat risiko pemeliharaan jaringan distribusi listrik secara cerdas. Aplikasi ini membantu mengalihkan paradigma pemeliharaan infrastruktur ketenagalistrikan dari reaktif menuju prediktif.

Fitur & Alur Utama:
1. Data Pemeliharaan: CRUD data penyulang (feeder) per bulan dan tahun dengan dukungan import massal file Excel (.xlsx).
2. FMEA Labeling: Penilaian manual atau otomatis untuk menghitung nilai RPN (Risk Priority Number = Severity x Occurrence x Detection) untuk menghasilkan label risiko (Rendah, Sedang, Tinggi).
3. Split Data: Pembagian data berlabel menjadi data latih (train) dan uji (test) menggunakan Stratified Split secara proporsional.
4. Algoritma KNN (K-Nearest Neighbor): Training model menggunakan machine learning core native PHP dengan konfigurasi nilai K, metrik jarak (Euclidean/Manhattan), dan pemilihan fitur.
5. Evaluasi Model: Analisis performa model menggunakan Confusion Matrix, Klasifikasi Report (Akurasi, Presisi, Recall, F1-Score), dan kurva K-Accuracy.
6. Laporan Risiko: Visualisasi tren RPN, grafik donat, grafik bulanan, serta perbandingan ketidaksesuaian (disagreement) antara prediksi KNN dan FMEA manual.

Tech Stack:
- Backend: PHP 8.1 (Custom MVC native tanpa framework)
- Database: MySQL (PDO prepared statements)
- Frontend: Tailwind CSS & Material Symbols Outlined Icons
- Charts: Chart.js
- Excel Parser: PhpSpreadsheet (via Composer)
